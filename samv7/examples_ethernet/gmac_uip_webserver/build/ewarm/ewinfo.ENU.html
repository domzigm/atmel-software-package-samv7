gmac_uip_webserver

Please refer to the uIP documentation for more information about the TCP/IP stack, the webserver example. By default, the example does not use DHCP. If you want to use DHCP, please: - Open file uip-conf.h and don't comment the line "#define UIP_DHCP_on". - Include uip/apps/dhcps to compile.
